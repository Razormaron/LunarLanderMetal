# Lunar Lander

A Lunar Lander game for macOS, built with Metal.

## Download

Grab the latest release from the [Releases page](../../releases) — unzip, drag the app anywhere, double-click to play.

> First launch: if macOS blocks it, right-click → Open → Open.

## Controls

| Key | Action |
|-----|--------|
| Space | Launch / Restart |
| ↑ or W | Main thruster |
| ← or A | Rotate left |
| → or D | Rotate right |

## Scoring

Land safely on a yellow pad to earn points. Chain consecutive landings without crashing to build your run score — that's your high score, saved between sessions.

## Build from source

Requires macOS 13+ and Xcode command-line tools.

```bash
git clone https://github.com/Razormaron/LunarLanderMetal
cd LunarLanderMetal
./install.sh
```
